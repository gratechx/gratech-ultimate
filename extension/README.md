
# Comet X Ultimate v4

إضافة متصفح (Chrome/Edge - MV3) بميزات:
- Sidebar فيروزي دائم يشبه Copilot.
- ملاحظات مع وسوم ذكية (اقتراحات أولية).
- إدارة تبويبات (تثبيت/تلوين باستخدام تخزين محلي كبديل).
- صفحة إعدادات لتخزين ghToken و ghRepo.
- Placeholder لاستيراد GitHub (Issues/PRs) ومزامنة سحابة لاحقًا.

## التحميل أثناء التطوير
1. افتح المتصفح (Chrome/Edge).
2. ادخل إلى `chrome://extensions` أو `edge://extensions`.
3. فعّل وضع المطور (Developer mode).
4. اختر "Load unpacked" وحمّل مجلد `cometx-ultimate-v4`.
5. اضغط على أيقونة الإضافة لفتح الـ Side Panel.

## ملاحظات
- تعتمد تلوين التبويب على تخزين قيمة مرتبطة بمعرّف التبويب (لا يوجد API لتلوين فعلي).
- اقتراح الوسوم مبسّط؛ سنستبدله لاحقًا بنموذج أقوى للفهرسة العربية.
- الاستيراد من GitHub متوقف في بيئة الأوفلاين لكنه مُمهد بإعدادات.

## الأمن
- لا تخزّن Tokens في النص الواضح خارج `chrome.storage`.
- لاحقًا سنضيف تشفيرًا ومزامنة سحابية اختيارية.
